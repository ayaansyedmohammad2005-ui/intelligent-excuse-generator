# Intelligent Excuse Generator

An AI-powered, context-aware excuse generator built using Python and Streamlit.
The system generates realistic excuses for work, school, and social situations
along with apology messages and emergency alerts.

## Features
- Context-based excuse generation (Work, School, Social)
- AI-style apology generator
- Emergency alert message creation
- Excuse history and favorites
- Multi-language support
- Simple and user-friendly interface

## Technologies Used
- Python 3
- Streamlit
